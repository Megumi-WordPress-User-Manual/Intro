Add, Edit, Delete Tags

Add Tag

1. Click "Posts", then "Tags".

2. Enter tag name.

3. Enter slug name.

4. Click "Add New Tag".

Edit Tag

1. Select "Posts" -> "Tags".

2. On the table on the right side, click on the Tag and select Edit, Quick Edit, Delete or View.
