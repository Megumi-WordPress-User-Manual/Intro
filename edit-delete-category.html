Edit / Delete a Category

1. Select "Posts" -> "Categories".

2. Choose a category on the table on the right side, and click "Edit".

3. Edit the category such as name, slug, parent and / or description and clcik Upgrade.

4. If you would like to delete a category, select a category on the table on the right and click "Delete".
